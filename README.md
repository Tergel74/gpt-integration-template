### Installation

```bash

npm install

# Setup environment
cp .env.example .env.local
# Add your API keys to .env.local

# Setup database
# 1. Create Supabase project
# 2. Run supabase/schema.sql in SQL Editor
# 3. Promote yourself to admin with your email
```
